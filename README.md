# my-bruno

This repository is a small reference project used to track the latest released version of [Bruno](https://github.com/usebruno/bruno).

The file `last_version.txt` stores the currently recorded latest version, so this repository can be used as a simple source of truth for checks, scripts, or lightweight automation.

Useful upstream links:

https://api.github.com/repos/usebruno/bruno/releases/latest

This GitHub API endpoint returns metadata for the latest Bruno release. It is useful when you want to read the most recent version programmatically.

https://github.com/usebruno/bruno/releases

This is the official Bruno releases page on GitHub, where you can browse release notes, download published versions, and verify the latest available release manually.
